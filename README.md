## 📌 What Is This Project?

This is not a collection of training slides.

This is a complete, end-to-end Learning and Development operating system — built for a fictional global MNC called Kartik Corp., operating across India, USA, and Europe with 312 employees across 6 departments.

Every single deliverable in this project exists because a specific finding in the data justified it. Nothing was built because it sounded like a good idea. Everything traces back to evidence.

## 🧠 The Problem This Project Solves

Most L&D portfolios show scattered outputs. A deck here. A certificate template there. A workshop agenda somewhere else. None of it connected. None of it justified by data. None of it proving it worked.

This project shows something different — a closed loop:

```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   🔍 DIAGNOSE  ──►  📐 DESIGN  ──►  🚀 DELIVER         │
│   Find the real     Build the fix    Make it            │
│   problem with      specific to      accessible         │
│   data              that problem     to everyone        │
│                                           │             │
│        ▲                                  ▼             │
│                                                         │
│   📊 MEASURE   ◄──  👂 LISTEN                           │
│   Prove it          Keep the system                     │
│   worked with       two-directional                     │
│   a dashboard       with feedback forms                 │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

Every piece in this project lives at one of those five points in the loop. Nothing exists outside it.

## 🔍 What the LNA Found

The Learning Needs Analysis identified 5 priority skill gaps across Kartik Corp.'s workforce:

| Priority | Skill Area | Score | Required | Severity |
|---|---|---|---|---|
| 🔴 1 | AI & Digital Literacy | 1.8 / 5 | 3.5 | CRITICAL |
| 🔴 2 | People Management | 2.4 / 5 | 4.0 | HIGH |
| 🟡 3 | Data-Driven Decision Making | 2.6 / 5 | 3.5 | MODERATE |
| 🟡 4 | Consultative Selling | 2.8 / 5 | 4.0 | MODERATE |
| 🟢 5 | DEI & Inclusive Leadership | 3.1 / 5 | 3.5 | LOW |

Every programme in this project was designed to close one or more of these gaps. Nothing was built without this justification.

## 👤 About This Project

Built by **Kartik Sharma**

This project was designed to demonstrate Senior L&D Specialist capability — not just the ability to create training content, but the ability to design, justify, deliver, and measure a complete learning system from the ground up.

**What this project demonstrates:**
- Diagnosing skill gaps before designing solutions
- Building programmes that are justified by specific findings
- Making programmes deliverable at scale through documentation
- Creating a single access point for all L&D resources
- Hosting real, completable digital learning
- Measuring impact through dashboards and feedback systems
- Connecting every piece into one coherent system

All company names, employee data, programme outcomes, and scenarios in this project are entirely fictional and created for portfolio demonstration purposes only.
